# mahoutinaction

Examples from the [Mahout in Action](http://www.manning.com/owen/) book, ported to Clojure

## Setup

1. Build and install
[Mahout](https://cwiki.apache.org/MAHOUT/buildingmahout.html)

2. Load the dependencies

```
lein deps
```

## Usage

```
lein run
```
